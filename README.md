# 通通变成马赛克

把普通视频转换成像素块/马赛克风格动画的小工具

![Python](https://img.shields.io/badge/Python-3.10%2B-3776AB)
![Platform](https://img.shields.io/badge/Platform-Windows-0078D4)
![License](https://img.shields.io/badge/License-MIT-111111)

</div>

导入视频，设置尺寸、马赛克块大小、帧数和 GIF FPS，工具会自动提取视频画面并生成马赛克帧。处理完成后可以在窗口里直接预览，也可以导出 GIF 动图或 PNG 截图。

## 快速使用

不想折腾环境的话，直接下载仓库里的 Windows 便携版：

**下载入口：** [通通变成马赛克_Windows便携版.zip](./通通变成马赛克_Windows便携版.zip)

解压后打开文件夹，双击运行：

```text
通通变成马赛克.exe
```

便携版已经打包好运行环境，不需要另外安装 Python。

## 主要功能

| 功能 | 说明 |
| --- | --- |
| 导入视频 | 支持常见视频格式，推荐 `MP4 + H.264` |
| 马赛克转换 | 将视频帧转换成像素块风格画面 |
| 实时预览 | 处理完成后直接在窗口里播放成品 |
| 导出 GIF | 把处理后的帧合成为 GIF 动图 |
| 导出 PNG | 保存当前预览画面为 PNG 截图 |
| 简洁界面 | 保留必要设置，操作流程尽量干净直接 |

## 使用流程

1. 打开 `通通变成马赛克.exe`
2. 点击 `选择视频`
3. 按需要调整参数
4. 点击 `开始处理`
5. 在窗口中预览成品
6. 选择导出 `GIF` 或 `PNG`

生成的文件会放在工具目录下的：

```text
mosaic_outputs
```

## 参数建议

| 参数 | 作用 | 推荐值 |
| --- | --- | --- |
| 尺寸 | 控制成品画面的最大边长度 | `480` |
| 马赛克块 | 控制像素块颗粒大小 | `8` 或 `10` |
| GIF FPS | 控制预览和 GIF 播放速度 | `24` |
| 提取帧数 | 控制从视频中抽取多少帧 | `120-240` |

如果电脑比较卡，可以先用这些轻量设置试效果：

```text
尺寸：360
马赛克块：10
GIF FPS：16
提取帧数：120
```

## 从源码运行

需要 Python 3.10 或更高版本。

安装依赖：

```bat
pip install -r requirements.txt
```

运行：

```bat
python mosaic_tool.py
```

## 支持格式

推荐使用：

```text
MP4 + H.264
```

工具也预留支持：

```text
.mp4
.mov
.m4v
.webm
```

不同视频编码的兼容性可能不同。如果某个视频无法处理，建议先转成普通 MP4 再试。
