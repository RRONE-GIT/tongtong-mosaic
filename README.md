# 通通变成马赛克

一个简洁的视频马赛克工具。导入视频后，工具会提取视频帧，把画面转换成像素块/马赛克风格动画，并在窗口里预览成品。处理完成后可以导出 GIF，也可以导出当前预览帧的 PNG 截图。

## 直接使用 Windows 便携版

仓库里包含：

```text
通通变成马赛克_Windows便携版.zip
```

下载后解压，打开解压出来的文件夹，双击：

```text
通通变成马赛克.exe
```

不需要单独安装 Python。推荐使用普通的 `MP4 + H.264` 视频。

## 从源码运行

需要 Python 3.10 或更高版本。

安装依赖：

```bat
pip install -r requirements.txt
```

运行工具：

```bat
python mosaic_tool.py
```

## 参数说明

- `尺寸`：控制成品画面大小，推荐 `480`，电脑较卡可以调低到 `360`。
- `马赛克块`：控制像素块颗粒大小，推荐 `8` 或 `10`。
- `GIF FPS`：控制预览和导出 GIF 的播放速度，推荐 `24`。
- `提取帧数`：控制从视频里抽取多少帧来制作动画，推荐 `120-240`。

## 输出

处理后的文件会生成在工具目录下的 `mosaic_outputs` 文件夹里。

## 文件说明

- `mosaic_tool.py`：工具源码
- `通通变成马赛克_使用说明.md`：使用说明书
- `通通变成马赛克_Windows便携版.zip`：Windows 便携版压缩包
