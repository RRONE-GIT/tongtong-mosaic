# 通通变成马赛克 - 第一次使用说明

这个工具可以把视频变成像素块/马赛克风格动画，并且可以预览成品，最后导出 GIF 或 PNG 截图。

## 1. 打开工具

打开解压后的工具文件夹。

双击这个文件：

```text
通通变成马赛克.exe
```

也可以在 cmd 里进入工具文件夹后运行：

```bat
通通变成马赛克.exe
```

打开后会出现一个窗口，窗口名字是：

```text
通通变成马赛克
```

## 2. 导入视频

点窗口里的：

```text
选择视频
```

推荐优先使用：

```text
MP4 视频
```

工具也预留支持：

```text
.mp4
.mov
.m4v
.webm
```

最稳的是普通的 `MP4 + H.264` 视频。

## 3. 参数怎么填

### 尺寸

控制成品画面的大小。

推荐：

```text
480
```

电脑比较卡可以调低：

```text
360
```

想更清楚可以调高：

```text
600 或 720
```

尺寸越大，处理越慢，导出的 GIF 也越大。

### 马赛克块

控制像素块颗粒大小。

推荐：

```text
8 或 10
```

参考：

```text
3-6    比较细，接近原视频
8-12   推荐，像素块明显，也还能看清内容
16-24  很粗，马赛克感强
30-40  超粗，容易变抽象
```

### GIF FPS

控制预览和导出 GIF 的播放速度。

推荐：

```text
24
```

电脑卡可以用：

```text
12 或 16
```

越高越顺滑，但 GIF 文件会更大。

### 提取帧数

控制从视频里抽多少张画面来做动画。

推荐：

```text
120-240
```

参考：

```text
60-120   更快，适合试效果
120-240  推荐，比较均衡
240-360  更顺滑，但更慢
500      很多帧，可能会比较卡
```

注意：现在这个数值不是自动识别原视频 FPS，而是把整个视频平均抽成你填的帧数。

## 4. 开始处理

选好视频和参数后，点：

```text
开始处理
```

工具会自动执行：

```text
提取视频帧 -> 生成马赛克帧 -> 准备预览 -> 播放成品
```

处理完成后，会自动在窗口里播放预览。

## 5. 预览

处理完成后可以点：

```text
暂停预览
```

再点一次会继续播放。

## 6. 导出

### 导出 GIF

点：

```text
导出 GIF
```

会把整个马赛克动画导出成一个 GIF 动图。

### 导出 PNG 截图

点：

```text
导出 PNG 截图
```

会把当前预览画面保存成一张 PNG 图片。

注意：PNG 不是导出所有帧，只是保存当前看到的这一帧。

## 7. 处理后的文件在哪里

每次处理都会自动生成一个输出文件夹：

```text
工具文件夹\mosaic_outputs
```

里面会保存：

```text
source_frames   原始提取帧
mosaic_frames   马赛克帧
```

你手动导出的 GIF 或 PNG，会保存到你在导出窗口里选择的位置。

## 8. 第一次推荐设置

第一次使用建议这样填：

```text
尺寸：480
马赛克块：8
GIF FPS：24
提取帧数：240
```

如果觉得卡，改成：

```text
尺寸：360
马赛克块：10
GIF FPS：16
提取帧数：120
```

如果只是先试试效果，改成：

```text
尺寸：360
马赛克块：8
GIF FPS：12
提取帧数：60
```

## 9. 常见问题

### 画面太抽象

把马赛克块调小，比如：

```text
8 改成 5
```

### 处理太慢

降低这两个：

```text
尺寸
提取帧数
```

比如：

```text
尺寸 480 改 360
提取帧数 240 改 120
```

### GIF 太大

降低：

```text
尺寸
GIF FPS
提取帧数
```

### 视频打不开

优先换成普通 MP4 视频再试。

比较稳的格式是：

```text
MP4 + H.264
```

## 10. 一句话流程

```text
打开工具 -> 选择视频 -> 用推荐参数 -> 开始处理 -> 看预览 -> 导出 GIF 或 PNG 截图
```
